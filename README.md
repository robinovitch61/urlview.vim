# urlview.vim

List and open URLs easily.

<GIF>

## Usage

```vim
:Urlview
```

Optionally, you can assign keybindings. For example:

```vim
nnoremap <silent> <leader>u :Urlview<CR>
```

## Installation

Pick your favorite plugin manager:

```vim
Plug 'strboul/urlview.vim'
```

No dependencies.
